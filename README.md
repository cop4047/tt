16-bit ADC Behavioural Model – Tiny Tapeout Project

This project implements a behavioural 16-bit Analog-to-Digital Converter (ADC) using Verilog. The goal is to explore mixed-signal modelling, quantisation, conversion delay, and switching-activity estimation within a digital simulation environment such as Tiny Tapeout.

Although Tiny Tapeout chips are fully digital, this project demonstrates how analog behaviour can be approximated and tested using Verilog simulation techniques.
